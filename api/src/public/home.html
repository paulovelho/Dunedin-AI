Dunedin AI API

JSON HTTP API for the Dunedin AI Kindle/web highlights manager. Users upload Kindle clipping files, search highlights and attach notes.

All endpoints are mounted under /api/v1. Authenticated requests must include a Firebase ID token as Authorization: Bearer <token>.
